The Path

I wanted the path to go aimlessly around the forest, representing how people often don't know where they're going in life when they rely on others for a route to follow, or something like that... ¯\_(シ)_/¯

Generate Path

This script generates a path given waypoints I have positioned in the forest using a path creator algorithm by Sebastian Lague.
I recommend watching his videos on it to understand how it works.

I wrote this script, and didn't use the nodes attached to the path, to position the path, as it allowed me to automatically make the path go up and down with the terrain, along with the rest of the scripts on this page.
