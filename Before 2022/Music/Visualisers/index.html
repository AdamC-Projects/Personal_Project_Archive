Visualisers

Music visualisers has always fascinated me, it's like the visuals are dancing around the screen, enjoying the music as much as I, and I can sit and watch it for hours ( ＾v＾) (...maybe)

Making my own music visualisers has taught me a lot about how one goes about decomposing a sound track, the structure of WAV files, and how mathematical the nature of audio analysis is.
They're also the most satisfying and cool projects when they work :D

All of my visualisers work in real time, because all the music I listen to is on youtube, so to capture the audio I use an app called Voicemeeter.

GIF to the Beat


This is the coolest code I've written!
The code gets audio data from the stream in real time; gets the bass from the data; subtracts it from the previous sample of data (to get the gradient of the sound wave); and iterates a given GIF by a number of frames proportiaonal to said gradient!
The idea being, wherever there is a steep gradient in time x bass amplitude graph, there has been a drum beat. In effect, this means the gif moves to the beat!
Due to the simplicity of the beat detection algorithm, this code works brilliantly on synthwave music, lofi, and any other songs with a distinct beat, but still decently for other songs. It also works best with seamless GIFs.
It threw me out of my seat how well this worked once I had written it!

GIF to the Beat - Wallpaper Engine


This project was the natural extension of the one above: of course I want a GIF to the Beat as my wallpaper!

Wallpaper Engine has a brilliant editing tool that allows you to create your own wallpapers using HTML and Javasctipt. Initially, I manually split my favourite GIF's (the cyberpunk coffee one) frames into PNGs and created a website that looped through each PNG in order. This worked well, and since I already knew how I would go about visualising the audio, it was relativley easy to code.
However, like with the python version above, I wanted to be able to animate any> GIF to the beat. This proved more difficult, as the only library I could find to do this was libgif.
I had to modify some of the code in it's JS file so that the image would:
• Fill the screen
• Align with the centre of the screen
• Have toggleable image smoothing

It was a fun weekend project, and I have uploaded the final product to the Steam Workshop - which you can access here.
