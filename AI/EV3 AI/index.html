About: EV3 AI

Since I aquired my first lego kit (at the age of too long ago for me to remember), I've always building my own contraptions, machines, robots, mainly with lego.
After using a Lego EV3 kit my school had, I decided to get myself an EV3 brick, so I could make more sophisticated robots at home.

However, I didn't that the vanilla EV3 made you use an oversimplified drag and drop programming language, so I followed a tutorial to install ev3dev onto my laptop and EV3 brick, so that I could read from the EV3 sensors and control the actuators remotely (from my laptop) in python.

This allows me to combine the EV3 robot with other sensors, such as a camera, increasing the robot's potential.

The only issue is, if I want to make a new robot, I'll need to deconstruct the previous one... which is why I intend to document all the final (not prototype) robot's here, so I don't forget about them!
Each robot has it's own page on the left.

The camera I'm using with the EV3

To save me explaining for each EV3 project I use a camera for, I'll explain here. I'm using a "hacked" YI-home camera, as this allows the camera to stream live footage onto a local server, from which I can grab images with python. To do this, I followed a tutorial here.
