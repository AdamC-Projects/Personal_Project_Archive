Logic gates

I used transistors, resistors and diodes to create all the logic gates.
The AND gate is, counterintuitively, composed of a NAND gate into a NOT gate.
The XNOR gate is composed of an OR and NAND gate into a NAND gate.
The XOR gate is, counterintuitively, composed of an XNOR gate into a NOT gate.

These were the logic gates I used in my 3-bit adder, below.
When I made my 2-bit adder, which I made last year in Feburary, I used bigger versions of the same gates.
